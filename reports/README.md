Per-run output: candidate comparison tables, the **invariant pass/fail matrix** (I1–I13 × model),
rank-recovery scores vs planted truth, and a human-readable per-game attribution example.

## The headline gate (post-pivot, 2026-06-24)

- **`real-eval.md`** — the project's **accuracy gate**: each model's agreement with the real MHR
  season (head-to-head gauntlet + published-rank Spearman ρ). Generated by `python -m harness.real_gate`.
- **`real-h2h.md`** — companion analysis: biggest MHR→gauntlet movers + giant-killer case studies.
- **`real-data-quality.md`** — the real dataset's data-quality report.

`comparison.md` (synthetic §7 rank recovery + invariant matrix) is now **diagnostic, not the gate** —
its score was partly an artifact of the old `base=3` floor's anchoring. The §7 scenarios still guard
I1–I13 as targeted unit tests. Walk-forward prediction (log-loss / calibration) is Stage-B Phase B4.
